# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1.0] - 2024-07-31

### Added

*   Initial release of the HA Zigbee Migration Tool.
*   Web UI for viewing ZHA devices and initiating migration.
*   Core migration logic for transferring Zigbee devices from ZHA to Zigbee2MQTT.
*   Automatic backup of the Zigbee2MQTT `database.db` file before migration.
